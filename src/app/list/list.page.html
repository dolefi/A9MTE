About A9MTE project @FAI UTB About This app was created by Filip Dolezal using Ionic framework as a project fo A9MTE @FAI UTB. App loads data from NewsApi.org (trending articles) and displays it in cards on main page, you can read whole article (opens browser) or store the article to memory. App is than able to load cards from stored memory on another page (clearing is possible) and uses ion storage module. User can select from which country should the news be loaded (approx 12 to choose from). User can also search those trending articles based on keywords (if empty -> all loaded, if no story can be loaded empty list is given)
